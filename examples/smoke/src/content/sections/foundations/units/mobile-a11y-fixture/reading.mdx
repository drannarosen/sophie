This stub chapter exists so the `mobile-chapter-a11y.spec.ts` e2e
regression guard has a content surface to scan at 375 px:

- one display-math equation, so the `rehypeKatexDisplayA11y` plugin
  fires and the resulting `.katex-display` scroll container can be
  asserted focusable + named (closes issue #192).
- one wide ASCII `<pre>` block, so the `html { overflow-x: clip }`
  defense can be asserted to prevent user-visible horizontal scroll
  (closes issue #187).

Authoring intentionally avoids tables (pre-existing
`scrollable-region-focusable` violation), GFM task-list checkboxes
(pre-existing `label` violation), and asides/callouts (pre-existing
`.margin-note` carve-out) so the spec asserts the WS A fixes
without conflating with unrelated baseline issues.

## Display math

The Schwarzschild radius of a non-rotating black hole:

$$
r_s = \frac{2 G M}{c^2}
$$

Inline math like $E = mc^2$ stays inline.
