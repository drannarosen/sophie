# Your first mission

:::{important} Status: pending Phase 3+
Missions exist in the schema today (see
[Content schema → Mission](../reference/content-schema.md)) but the
authoring flow earns its keep alongside the AI-authoring kit in
Phase 3. Until then, this is a stub.
:::

## What you'll build (planned)

A mission tied to an existing textbook chapter:

- A scenario that frames a concrete pedagogical goal.
- A sequence of mission steps: prediction → demo-interaction →
  inference-prompt → reflection.
- Demo integration via the
  [Cosmic Playground manifest](../how-to/integrate-cosmic-playground.md).
- Optional Canvas export configuration.
- Audit verification: the mission references real concepts/skills,
  the demo exists, the prediction has a paired solution if needed.
